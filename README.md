# 📘 JavaScript Level 1 Projects (6–10)

This repository contains my next set of **beginner-friendly JavaScript projects**, focusing on programming logic, DOM manipulation, user interaction, and building interactive web applications.

These projects are part of my structured **50 JavaScript Projects Roadmap**, designed to strengthen my front-end development skills step by step.

---

## 🚀 Projects Included

### 6️⃣ Temperature Converter
- Convert temperatures between Celsius, Fahrenheit, and Kelvin
- Real-time conversion based on user input
- Demonstrates mathematical calculations and DOM manipulation

### 7️⃣ Number Guessing Game
- Generate a random number
- Allow users to guess until they find the correct number
- Uses loops, conditions, `Math.random()`, and event handling

### 8️⃣ Basic Calculator
- Perform addition, subtraction, multiplication, and division
- Interactive calculator interface
- Demonstrates functions, event listeners, and DOM manipulation

### 9️⃣ To-Do List App
- Add new tasks
- Delete completed tasks
- Manage daily task list dynamically
- Demonstrates arrays, DOM manipulation, and UI updates

### 🔟 Simple Form Validation
- Validate user inputs
- Email format validation
- Password validation
- Display error messages dynamically
- Demonstrates regular expressions (Regex), conditions, and DOM manipulation
